# Attendance Portal

## Overview

Attendance Portal is a focused React + Tailwind single-page application for club attendance management. It provides a members dashboard, a meeting scheduler, an intuitive attendance-marking UI, and consolidated attendance summaries so club organisers can quickly see participation trends and per-member activity. Deployed at: https://mdc-attendance-portal.vercel.app/

## Core Features

- Members Dashboard: create, edit, and view member profiles and participation history.
- Meeting Scheduler: create meetings/events with date, time, and notes.
- Attendance Marking: fast UI for marking attendees per meeting (individual and bulk).
- Attendance Summaries: per-member and per-event summaries and simple export-ready views.
- Admin Dashboard: manage members, meetings, and attendance records from a central admin view.
- Responsive UI: Tailwind-powered layout for desktop and mobile.

## Tech Stack

- Frontend: React (Create React App)
- Styling: Tailwind CSS
- Languages: JavaScript, HTML, CSS

## Requirements

- Node.js 16+ and npm

The app is deployed at: https://mdc-attendance-portal.vercel.app/

## Project Structure

- `src/` — application source (components, styles)
- `public/` — static assets and `index.html`
- `build/` — production build artifacts (generated)
- `package.json` — scripts and dependencies

## Workflows

- Add members: use the Members Dashboard to add or import members.
- Schedule a meeting: create a meeting/event from the Meetings view.
- Mark attendance: open the meeting, mark attendees quickly, save the record.
- View summaries: open the dashboard to view participation counts and trends over time.
